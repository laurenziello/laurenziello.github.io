Elasticsearch

Elasticsearch

Elasticsearch è un RESTful motore di ricerca distribuito.

Config

Scaricare elaticsearch da https://www.elastic.co/downloads/elasticsearch.

Run

Eseguire il comando

 bin/elasticsearch

Sotto windows

 bin/elasticsearch.bat

Il motore ora sarà in ascolto sulla porta 9200.
