SetUp

Installazione

Scaricare nuget.exe Windows x86 command line dal sito https://www.nuget.org/downloads

Creare una nuova cartella in C:\Program Files\Nuget e copiare all'interno il file scaricato.

Aggiungere C:\Program Files\Nuget alla Variabile di sistema Path.

Eseugire il comando nuget da powershell o cmd per testare il funzionamento.

Aggiungere Origine Pacchetti

Eseguire il comando da powershell

nuget sources add -name "mst" -source //[ipmacchina]/DATI/Reparto_IT/Nuget -username betpz\.... -password ....

per controllare che l'origine dei pacchetti sia stata aggiunta correttamente eseguire il comando

nuget sources

tra le fonti registrate è presente mst con la dicitura [Abilitato].

Ora in visual studio in gestione pacchetti NuGet è possibile scegliere nella dropdown di Origine pacchetto oltre a nuget.org anche mst. Cambiando la selezione è possibile visualizzare i pacchetti nuget presenti nella cartella condivisa.
